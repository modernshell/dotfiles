Vim est un éditeur de texte  terminal unix, doté de nombreux raccourcis clavier mais qui s’oublient facilement.


Des aide-mémoires sont disponibles sur le web mais ceci ne sont modifiables.
Les fonctions récentes sont souvent manquantes. 

L'objet est de produire un Aide mémoire 
sur les raccourcis claviers des fonctions Vim (vi, nvim)


Liste des exigences 
méthode MoSCoW

 M1: la fiche doit rappeler les raccourcis de fonctions
 M2: la fiche doit être facile a utiliser et concise 
 M3: la fiche doit pouvoir être imprimable en petit format 
 M4: les raccourcis de fonctions doivent être groupés 

 S1: la fiche devrait contenir un lien vers les données sources
 S2: les données sources devraient être modifiables
 S3: les données sources devraient être versionnées
 S4: les données sources devraient être testées

 C1: la fiche pourrait contenir un supplément de liens utiles
 C2: la fiche pourrait aussi concerner les greffons

 W1: la fiche ne concerne que vim ou greffons
 W2: la fiche n'est pas un tutoriel mais un rappel des raccourcis de fonctions
